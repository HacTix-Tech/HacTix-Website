WiFi Login Logger

The Break Down...

The HacTix - Tech WiFi Login Logger, works by creating a decoy WiFi network that you control. This can appear as a real network on nearby devices, and when connected, it will open a login page that prompts the user for credentials. You can customize this page to look however you want. Once the user logs in to this page, their login credentials will be logged on the logger device, where you can retrieve them. You need a USB Type-C cable to connect to your laptop to access these logins. We encourage you to modify our products, improve on them, and tailor them to your needs!

The device is powered by MicroPython, (a lightweight version of Python meant for microcontrollers and dev boards). We chose MicroPython so that the device can easily be MODED.
